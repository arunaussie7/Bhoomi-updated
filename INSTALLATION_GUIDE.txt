üîç CHECKING PYTHON INSTALLATION
===============================

Step 1: Verify Python is installed
---------------------------------
Open Terminal/Command Prompt and type:
  python --version
  OR
  python3 --version

If you see a version number (3.7 or higher), Python is installed.
If you get an error, download Python from: https://python.org

Step 2: Install Python (if not installed)
-----------------------------------------
1. Go to https://python.org
2. Click "Downloads"
3. Download the latest version for your OS
4. Run the installer
5. IMPORTANT: Check "Add Python to PATH" during installation
6. Restart your computer after installation

üîë GETTING YOUR COHERE API KEY

-----------------------------
1. Go to https://cohere.ai
2. Click "Sign Up" or "Get Started"
3. Create a free account

Step 2: Get Your API Key
------------------------
1. Log in to your Cohere account
2. Go to the "API Keys" section
3. Click "Create API Key"

Step 3: Update the API Key
--------------------------
After running the setup, edit the .env file:
1. Open the .env file in your project folder
2. Replace the existing API key with your own:
   COHERE_API_KEY=your_actual_api_key_here
3. Save the file
4. Restart the application

‚ö†Ô∏è TROUBLESHOOTING
==================

PROBLEM: "Python not found" error
---------------------------------
SOLUTION:
1. Download Python from https://python.org
2. Install with "Add to PATH" option checked
3. Restart your computer
4. Try running the setup again

PROBLEM: "pip not found" error
------------------------------
SOLUTION:
Try these commands in order:
1. python -m pip install -r requirements.txt
2. python3 -m pip install -r requirements.txt
3. pip3 install -r requirements.txt

PROBLEM: Permission errors (Mac/Linux)
-------------------------------------
SOLUTION:
1. Make the script executable: chmod +x start_app.sh
2. Then run: ./start_app.sh

PROBLEM: Browser doesn't open automatically
------------------------------------------
SOLUTION:
1. Open your web browser manually
2. Go to: http://localhost:8501

PROBLEM: "Port already in use" error
------------------------------------
SOLUTION:
1. Stop any other applications using port 8501
2. Or use a different port: streamlit run app.py --server.port 8502

PROBLEM: Package installation fails
----------------------------------
SOLUTION:
1. Update pip: python -m pip install --upgrade pip
2. Try installing packages one by one:
   pip install streamlit
   pip install PyPDF2==3.0.1
   pip install cohere
   pip install python-dotenv
   pip install streamlit_extras

PROBLEM: Cohere API Key issues
------------------------------
SOLUTION:
1. Get your free API key from https://cohere.ai
2. Edit the .env file in your project folder
3. Replace the API key with your own:
   COHERE_API_KEY=your_actual_api_key_here
4. Save the file and restart the application

PROBLEM: "API key not found" or "Invalid API key" errors
--------------------------------------------------------
SOLUTION:
1. Check that the .env file exists in your project folder
2. Verify your API key is correct in the .env file
3. Make sure there are no extra spaces or characters
4. Restart the application after making changes
